introduction au CSS

Introduit au milieu des années 1990, CSS devient couramment utilisé dans la conception de sites web et bien pris en charge par les navigateurs web dans les années 2000.

Le CSS est un language de style qui définit la présentation des documents HTML. Par exemple, le CSS couvre les polices, les couleurs, les marges, les lignes, la hauteur, la largeur, les images d'arrière-plan, les positionnement évolués et bien d'autres choses.

Le but d'avoir le css dans vos sites c'est de compenser les manques du language HTML en ce qui concerne la mise en page et la présentation
