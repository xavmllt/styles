Les feuilles de styles permettent le formatage et la mise en page Web. Introduit au milieur des années 1990, CSS devient couramment utilisé dans la conception de sites web et bien pris en charge par les navigateurs web dans les années 2000. En général le language HTML soit destiné à décrire le contenu, les feuilles de style sont déstinées à décrire la forme du document. Le but d'avoir le css dans vos sites c'est de compenser les manques du language HTML en ce qui concerne la mise en page et la présentation. Le CSS est un language de style qui définit la présentation des documents HTML. Par, exemple, CSS couvre les polices, les couleurs, les marges, les lignes, la hauteur, la largeur, les images d'arrière-plan, les positionnements évolués et bien d'autres choses.
